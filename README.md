# Lexicon - Vocabulary Learner

A beautiful, interactive React app to learn 100 English words across 10 difficulty levels.

## Features

- **📖 Flashcard Mode**: Flip cards to reveal definitions and etymology
- **⚡ Quiz Mode**: Test yourself with multiple-choice questions
- **🎯 10 Difficulty Levels**: From foundational to lexical mastery
- **🔥 Streak Tracking**: Monitor your learning progress
- **🌱 Etymology**: Learn word roots and origins
